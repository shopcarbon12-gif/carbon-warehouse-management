Senitron-Related Files Audit (Handheld)

Date: 2026-04-14

Device package: com.senitron.senitron_chainway_handheld_app

Executive Summary

  • Senitron is a Flutter-based Android app with native UHF support libraries embedded in its APK.
  • The device stores rich operational output in shared storage under /sdcard/SenitronRFID, including inventory CSV exports and metadata JSON files.
  • Scanner behavior appears configurable by XML in shared storage under /sdcard/Scanner/data.
  • Private app data directories were not readable due lack of root / run-as restrictions.

Key Artifacts Found

1) Installed APK

  • APK path: /data/app/com.senitron.senitron_chainway_handheld_app-.../base.apk
  • Version: 2.123.0 (from package metadata)
  • Embedded native libs include:
    • libDeviceAPIM.so
    • libDeviceAPIQ.so
    • libIDFingerprintAlg.so
    • libflutter.so, libapp.so
  • This strongly indicates direct integration with OEM device APIs, not just generic keyboard wedge use.

2) Senitron Data Exports

  • Folder: /sdcard/SenitronRFID
  • Subfolders:
    • Inventory/ contains many session CSV files.
    • Data/ contains paired -Info, -J1, -J2 files in JSON and DAT formats.
  • Sample -Info.json contains session summary fields:
    • user_name, datetime, items_count, assets_count, note
  • Sample inventory CSV row format:
    • EPC,timestamp,zone/bin

3) Scanner Configuration Files

  • Folder: /sdcard/Scanner/data
  • Files:
    • KeyboardHelperParam.xml
    • MOTOParameters.xml
  • KeyboardHelperParam.xml includes useful keys:
    • scanner_etBroadcast=com.scanner.broadcast
    • scanner_etBroadcastKey=data
    • scanner_target=2 (broadcast path)
    • scanner_uhf_power=30
  • This confirms practical scanner broadcast contracts and defaults used on this handheld.
