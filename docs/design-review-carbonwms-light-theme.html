CarbonWMS light theme — unified direction

This document merges your scheme.png palette with DESIGN (2).md surfaces and spacing ideas. Clarification (your pic1 — Inventory catalog): “Not colorful” was not a request to grey-out the product. It meant: don’t add extra decorative treatment (e.g. teal gradients from an early mock). The canonical action style is the catalog toolbar: solid green (New / Export), blue (Import), slate (dropdowns like Lightspeed), orange (More) — white text, no gradients. Light-theme work should make every page use that same pattern so the app looks consistent. Fonts: keep existing CarbonWMS families; only size/weight/contrast as needed.
