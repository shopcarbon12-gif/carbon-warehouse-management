4. Security, OTA, and settings

  • Device ID — Android ID for deviceId on APIs.
  • Boot sequence — Chainway vs Zebra autodetect.
  • Power slider — Dashboard header; persists and reapplies to scanner.
  • OTA dialog — Dismiss-only on boot; install from header.
  • OTA polling — fetchMobileStatus every 3 minutes.
  • Release signing — Debug signing for internal QA; not Play Store ready.

6. Hardware & Geiger (RSSI) fidelity

  • Zebra — API3 tag RSSI when linked.
  • Chainway — CarbonChainwayRfidController extracts RSSI via reflection when the vendor object exposes it; falls back to -55 only when unknown.
  • Stubs — SIMULATE buttons for QA if hardware not linked.

7. Ghost filter & edge streaming

  • Server-driven visibility — postEpcVisibility before session/edge.
  • Geiger exception — GEIGER_FIND bypasses ghost queue.
  • Docs drift — Prefer server semantics is_visible_to_scanner in new comments.

8. Dependencies & native bundles
