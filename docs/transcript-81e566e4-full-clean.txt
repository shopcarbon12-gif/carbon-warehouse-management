The Matrix screen is not showing “how many Lightspeed says you have in stock.”

It’s built around RFID / WMS inventory:

- Columns like EPCs / active EPC count come from tags you’ve actually recorded in this system for that warehouse (the `items` table — what’s in stock in WMS).

Short version:  
- Sync = “copy the product list from Lightspeed.”  
- EPC counts = “what we know is in the building from RFID workflow,” not “what Lightspeed’s qty is.”

### If that password doesn’t work
