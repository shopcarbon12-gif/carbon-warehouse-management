📱 Mobile App

Locate Tag (Geiger Mode)

A "hot/cold" tag locating tool named after the Geiger counter concept. Staff enter or scan an EPC code, then physically walk around the warehouse while the handheld reads the signal strength of that specific tag. The beep frequency (and visual indicator) increases as the staff member gets closer to the item, guiding them to its exact location.

Target EPC Input

EPC / Tag Search Field
Text Input
A field where the staff member enters the EPC code of the specific tag they want to find. This can be entered by typing, pasted from a copied value, or potentially pre-filled if navigating from an item lookup result. Once set, this becomes the "target EPC" that the scanner will listen for specifically.
Suggestion: Typing a full EPC code (a long hex string like "E28011606000020700F7A6DB") is error-prone. The Locate Tag screen should be reachable directly from an item's detail view (Item Lookup, Inventory list) via a "Find this tag" button that pre-fills the EPC automatically. Manually typing an EPC just to use this screen is impractical on a mobile keyboard.

Geiger Proximity Interface

Hold-to-Scan Button
Hold Button
A button that must be held down to continuously scan. While held, the RFID reader transmits continuously and listens for the target EPC. As soon as the target EPC is detected, the proximity feedback starts. Releasing the button stops scanning. This "hold to scan" pattern prevents constant RFID transmission when the feature is not in use.
Suggestion: The hold-to-scan interaction is appropriate for power management but may be tiring during a long search (holding a button while walking through a warehouse). Consider a "continuous scan" toggle as an alternative — once enabled, scanning runs until stopped. For a quick locate, hold-to-scan is fine; for searching a large floor, continuous mode is better.
RSSI Signal Strength Display
Signal Indicator
Shows the raw RSSI (signal strength) value from the RFID antenna when the target tag is detected. The RSSI value ranges from a large negative number (far away, e.g., -85 dBm) to a smaller negative number (close, e.g., -45 dBm). Higher values mean closer proximity. The display updates in real time as the handheld moves.
Suggestion: The raw RSSI number (e.g., "-63 dBm") is meaningless to warehouse staff. Replace or supplement it with a visual proximity indicator — a signal strength bar (like Wi-Fi bars), a color gradient from cold blue (far) to warm red (close), or simply a large text label: "Getting warmer..." / "Very close!" / "Right there!" The Geiger beeping handles audio cues, but the visual should be equally intuitive.
Geiger Beep Audio
Audio Feedback
The app plays an audio beep that increases in frequency as the target tag gets closer — like a Geiger counter. At a distance, beeps are slow (once every few seconds). Very close to the tag, beeps become rapid and nearly continuous. The RSSI value is mapped to a beep interval using a proximity formula. This audio feedback allows staff to navigate by sound without constantly looking at the screen.
Suggestion: The Geiger audio is an excellent, intuitive design choice for this type of tool. One improvement: add a haptic (vibration) feedback option as well, since warehouse environments are often noisy and staff may have earbuds in or noise-canceling hearing protection. Also, ensure the audio beep works when the device is in silent mode — silent mode should only affect notification sounds, not active operational tools.

Overall Locate Tag Suggestions

1. No "navigate from item detail": The biggest friction is starting a locate session — staff need to know the EPC before they can use this tool. Adding a "Locate this item" button on every item detail card (in Item Lookup, inventory lists, cycle count mismatches) would make this tool dramatically more useful.

2. No visual proximity map: Even a simple floor plan or grid view showing the relative direction of the tag would be a significant upgrade. RFID direction-finding is technically limited, but RSSI strength could at least indicate "getting warmer" with a simple arrow or radial gradient.

3. Tag not found handling: If the staff member scans an entire area and the target tag is never detected, the app should acknowledge this clearly: "Tag not detected in this area — it may be too far away, blocked, or its status may indicate it is not in active inventory." A fallback suggestion helps rather than leaving the staff member wondering if the tool is broken.
