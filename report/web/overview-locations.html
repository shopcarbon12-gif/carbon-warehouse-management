Locations & Bins Manager

Where you manage all warehouse locations and the bins within each location. View, add, edit, and delete bins. Each bin shows its current in-stock item count. Also accessible via the old "/locations" URL which redirects here.

Location Selector

Location List / Tabs
Selector
A list or tab strip showing all warehouse locations. Clicking one selects it and loads that location's bins. The first location is selected by default. If a previously selected location is deleted, it resets to the first available.
The contextual location selector — where picking a location immediately loads its bins — is a clean and efficient layout for a multi-location warehouse. It avoids deep nested navigation.
Show a total bin count badge next to each location name: "Warehouse A (12 bins)." For tenants with many locations, add a search/filter within the location list.

Bins Grid

Bin Cards / Grid
Grid Display
Shows all bins for the selected location as cards. Each bin displays its code, name, and current in-stock item count. The layout is a responsive grid.
Showing the current item count on each bin card gives admins immediate inventory context while managing bins — this is exactly the right information to display here.
All bins look identical regardless of how many items they have. A bin with 0 items and a bin at full capacity are styled the same, giving no visual signal about inventory health per bin.
Color-code bin cards: green for normal, amber for nearly empty (needs replenishment), red for 0 items or over-capacity. Make the item count the most prominent number in the card design.
Add Bin Button
Primary Button
A button with a plus icon. Only active when a location is selected. Opens the Bin Editor Drawer to create a new bin for the selected location.
The add button is contextually tied to the selected location — it only creates bins for the currently active location, preventing the common mistake of adding a bin to the wrong location.
When no location is selected, visibly disable the button and show a tooltip: "Select a location first to add a bin." This removes ambiguity about why the button may not work.
Edit & Delete Bin Buttons (per bin)
Icon Buttons
Pencil icon opens the Bin Editor Drawer pre-filled for editing. Trash icon deletes the bin. Both presumably require admin permission (canCleanBins). Pencil icons are small.
Deleting a bin that still has items assigned to it is allowed without a warning about what will happen to those items. Deleting a bin with 47 items could leave those items without a bin assignment — floating at location-level only.
The edit and delete icons are small touch targets. On touch screens or for staff with large hands, tapping the correct icon reliably is difficult.
Block deletion of bins that still contain items, or show a specific warning: "This bin has 47 active items. Deleting it will leave them without a bin assignment. Consider moving items first." Also increase the tap target size for edit/delete icons — minimum 44x44px.

Bin Editor Drawer

Bin Editor (Add / Edit)
Side Drawer
A slide-in panel for adding or editing a bin. Fields include bin code and name. Saving creates or updates the bin. Closing without saving discards changes.
There is no validation shown if a bin code already exists when creating a new bin. The operator only finds out about the conflict after submitting.
Validate the bin code in real time as the user types — show an inline message if the code is already taken: "Bin code 'A4' is already used in this location." For new bins, suggest the next code in the sequence based on existing bins (e.g., if A1, A2, A3 exist, suggest A4).

Overall Assessment

The Locations & Bins Manager has a clear and logical layout. The contextual location selector, real-time item counts, and permission-gated edit controls are all solid design decisions for managing warehouse physical structure.
Key improvements:
1. Color-code bin cards by inventory health (0 items = red, low = amber, normal = green).
2. Block or clearly warn before deleting bins that still contain items.
3. Add real-time bin code validation in the editor drawer.
4. Increase touch target size on edit/delete icons.
