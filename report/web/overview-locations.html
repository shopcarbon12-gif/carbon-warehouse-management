Locations & Bins Manager

Where you manage all warehouse locations and the bins within each location. Lets you view, add, edit, and delete bins. Each bin shows its current in-stock item count. Also accessible via the old "/locations" URL which redirects here.

Location Selector

Location List / Tabs
Selector
A list or tab strip showing all warehouse locations (e.g., "Warehouse A," "Store B"). Clicking one selects it and loads that location's bins. The first location is selected by default on page load. If a previously selected location is no longer in the list (e.g., deleted), it resets to the first available.
Suggestion: For tenants with many locations, the list could become very long. Consider adding a search/filter within the location list. Also show the total bin count per location as a small badge: "Warehouse A (12 bins)."

Bins Grid / List

Bin Cards / Grid
Grid Display
Shows all bins for the selected location as cards or list items. Each bin displays its code, name, and current in-stock item count. The layout is a grid that adapts to screen size.
Suggestion: The bin count is the most important information — make it large and prominent. Consider color-coding bins: green for normal, yellow for nearly empty (needs replenishment), red for overloaded or 0 items.
Add Bin Button (+ icon)
Button
A button with a plus icon and label (possibly "Add bin" or PackagePlus icon). Only active when a location is selected. Clicking opens the Bin Editor Drawer to create a new bin for the selected location.
Suggestion: The add button is contextual (only works when a location is selected) — if no location is selected, it should be visibly disabled with a tooltip explaining why.
Edit Bin Button (pencil icon, per bin)
Button
A small pencil icon on each bin card. Clicking opens the Bin Editor Drawer pre-filled with that bin's data for editing. Only visible to users with the "canCleanBins" permission (admin role).
Suggestion: The edit icon is small. On touch screens and for warehouse workers, larger tap targets would be more reliable. Consider showing the edit button more prominently in a bin details panel that opens on click.
Delete Bin Button (trash icon, per bin)
Button
A trash icon on each bin card. Presumably deletes the bin after a confirmation prompt. Only visible to authorized users.
Suggestion: Deleting a bin that still has items assigned to it should either be blocked or show a warning: "This bin has 23 active items. Deleting it will leave those items with no bin assignment. Are you sure?" Permanent deletions should always require confirmation.

Bin Editor Drawer

Bin Editor (Add / Edit)
Side Drawer / Modal
A slide-in panel (drawer) from the side that shows a form for adding or editing a bin. Fields include bin code and name. Saving creates or updates the bin and refreshes the list. Closing the drawer without saving discards any changes.
Suggestion: The drawer should have clear "Save" and "Cancel" buttons with distinct styling. Also add validation feedback — if a bin code already exists, show an error before the user submits. For new bins, suggest a code format based on existing bins (e.g., if bins are "A1, A2, A3," suggest "A4" for the next one).
