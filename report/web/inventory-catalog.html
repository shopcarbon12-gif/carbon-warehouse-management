Inventory Catalog

The main product catalog page. Shows items synced from Lightspeed (SKU, name, vendor, price, stock counts) alongside a separate RFID view showing which tags exist in the system. Supports search, pagination, CSV export, and manual item entry.

Page Header Area

Page Title & Description
Header
Shows "Inventory catalog" with a subtitle explaining the page contains Lightspeed item lines and a separate RFID view. Admin users see additional buttons; regular users see a more limited interface.
Suggestion: The subtitle is technical. Replace with plain language: "Browse your product catalog and see which items have active RFID tags."
"Legacy inventory" Link
Link
A small teal link in the top-right that goes to the older flat inventory table. Marked implicitly as legacy.
Suggestion: If the legacy view is being phased out, this link should include a note "(old version — will be removed)" or simply be removed. Showing both creates confusion about which one to use.

Tabs: Lightspeed vs RFID View

Lightspeed Tab
Tab
Default view. Shows the product catalog synced from Lightspeed POS: item name, SKU, UPC barcode, vendor, color, size, retail price, and POS on-hand quantity. Paginated — 50 items per page. Has a search box that filters by name or SKU as you type (with a small delay). Clicking a row opens a detail modal.
Suggestion: The table has many columns. On smaller laptop screens, users have to scroll horizontally. Consider letting users choose which columns to show/hide, or collapsing less-used ones (like UPC) by default.
RFID Tab
Tab
Shows the RFID-specific view — individual EPC tags associated with items, their current location, bin, and status. Different data from the Lightspeed tab: here you see individual physical tags, not just SKU counts.
Suggestion: It would be very useful to be able to switch between these two views while keeping the same search query — e.g., search for "Black Dress" in the Lightspeed tab, then switch to RFID to see which physical tags exist for that item.

Search & Filter

Search Box
Search Input
A text field above the table. As you type, there is a short delay (debounce) and then the table updates to show only matching items. Matches on item name or SKU. Search state is maintained when you switch pages.
Suggestion: When there are no results for a search, the message should be more helpful: "No items found for 'blue dress'. Try a different search term or check if the catalog has been synced recently." Also, searching by UPC/barcode would be useful for scanning barcodes directly into the search field.

Action Buttons (Admin Only)

Lightspeed Sync Button (dropdown)
Button with Dropdown
A button with a down arrow that opens a menu with sync options. Allows admins to manually trigger a Lightspeed catalog sync directly from this page. Shows sync status/results after running. Only visible to Admin role users.
Suggestion: The sync takes time and there is no progress indicator during the sync. A spinning animation or progress message would reassure users it is working. Also, showing "Last synced 2 hours ago" near this button would help users decide if they need to sync.
Export CSV Button
Button
Downloads the currently visible catalog items as a CSV file. The file is named with today's date (e.g., "lightspeed-catalog-2026-04-11.csv"). Contains: Item Name, SKU, UPC, Vendor, Color, Size, Retail Price, Quantity (LS).
Suggestion: The export downloads only what matches the current search/filter. The filename and a count of exported rows should be confirmed to the user (e.g., "Downloading 47 items..."). Also consider letting users choose columns to include in the export.
Add New Item Button
Button
Opens an inline form to manually add a product to the catalog (for items not in Lightspeed). Fields include: Matrix UPC, Description, SKU, Vendor, Color, Size.
Suggestion: Manually adding items that then get overwritten on the next Lightspeed sync would be confusing. A warning should be shown: "Note: manually added items may be overwritten if they conflict with the next Lightspeed sync." Also clarify if these items will appear in Lightspeed or only in the WMS.
Import Button
Button
Allows bulk importing of catalog items, presumably from a CSV or similar format. The exact UI for this is within the "More" dropdown menu.
Suggestion: Provide a downloadable template CSV so users know exactly what format is expected before uploading. A preview of what will be imported before committing the import would also prevent mistakes.

Pagination

Pagination Controls
Navigation
Shows 50 items per page. Previous/Next page buttons let you navigate through pages. The current page number is shown. When you search, it resets back to page 1.
Suggestion: Consider showing "Showing 51–100 of 1,247 items" so users know how deep they are in the catalog. Also, allowing users to change the page size (25/50/100 per page) would be helpful for power users doing catalog reviews.

Item Detail Modal

Row Click → Detail Modal
Modal Popup
Clicking any row in the Lightspeed tab opens a detail panel (likely a modal or drawer) showing full item details including all attributes, price history, and associated RFID tags for that SKU.
Suggestion: The modal should allow editing item details (for admin users), not just viewing them. Also, a direct link to "see this item in Lightspeed" would be useful for cross-referencing.
