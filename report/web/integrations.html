Integrations

A read-only table showing all external systems connected to this tenant (e.g. Lightspeed POS). Shows current connection status, which location is linked, and when the last successful sync occurred.

Integrations Table

Provider Column
Table Column
Shows the name of the connected system (e.g., "lightspeed"). Text is automatically capitalized for display.
Suggestion: Show provider logos alongside text names for quick visual recognition. Also consider showing a colored status dot (green = connected, red = error, grey = not configured).
Status Column
Table Column
Shows the current status of the connection as raw text (e.g., "active", "error", "pending"). No color coding.
Suggestion: Status values should be color-coded: green badge for "active," red for "error," yellow for "pending." Raw text status values are easy to miss.
Location Column
Table Column
Shows which warehouse location the integration is linked to (e.g., "STORE-A"). If the integration is at the tenant level (not location-specific), it shows "tenant."
Suggestion: "tenant" is an internal term. Replace with "All locations" for clarity.
Last OK Column
Table Column
Shows the timestamp of the last successful API response from the integration (format: 2026-04-11T10:30:00). Shows "—" if never succeeded.
Suggestion: Show relative time ("2 hours ago") alongside the full timestamp. If "Last OK" was more than 24 hours ago, highlight in amber as a warning that the connection may be stale.
Last Job Column
Table Column
Shows the timestamp of the last sync job that ran (not necessarily successful). Helps diagnose if jobs are queued but never completing.
Suggestion: Adding a "Last job result" column (success/failure) would make this table much more actionable. Currently you can see a job ran but not if it succeeded.
Empty State
Empty State
When no integrations are configured, shows: "No integration rows — connect Lightspeed or seed data."
Suggestion: Replace developer language with a helpful call-to-action: "No integrations configured. Go to Settings → General to set up your Lightspeed connection." Include a direct link.

Overall Integrations Page Suggestions

1. Read-only is limiting: Admins should be able to test the connection, force a sync, or disconnect an integration directly from this page rather than having to navigate to Settings.

2. Add action buttons: "Test connection," "Sync now," and "View sync history" would make this page far more useful.

3. No page refresh: The page loads integration data once and does not auto-refresh. A manual refresh button or auto-refresh every 60 seconds would be useful for monitoring connection health.
