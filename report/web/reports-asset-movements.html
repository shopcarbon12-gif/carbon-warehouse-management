Asset Movements Report

A report showing all recorded movements of RFID-tagged items — when items changed location, moved between zones, or were transferred between bins. Useful for tracing the physical journey of any item through the warehouse.

Movements Table

Asset Movements Table
Data Table
Shows movement events with EPC, item details, source location/bin, destination location/bin, type of movement (transfer, scan, etc.), and timestamp. Read-only.
Suggestion: Add filtering by EPC or SKU (to trace one specific item), by date range, by movement type, and by location. Without filters, this report becomes unusable at scale. Also add a CSV export button.
Report Toolbar
Toolbar
The shared report toolbar component — typically includes date range picker, search, and export options depending on the workspace.
Suggestion: Make sure the toolbar has a date range filter that defaults to "Today" so the report opens with relevant data rather than loading everything from the beginning of time.
